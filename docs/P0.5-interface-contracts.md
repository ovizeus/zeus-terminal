### `calcConfluenceScore()`
**File**: `public/js/brain/confluence.js`  
**Category**: BRAIN — score computation

| | |
|---|---|
| **Params** | none |
| **Reads** | `S.signalData` (.signals[], .bullCount, .bearCount), `S.rsi` (5m), `S.ls` (.l, .s), `S.fr`, `S.oi`, `S.oiPrev` |
| **Writes** | `BM.confluenceScore` ← number(0-100), `CORE_STATE.score`, `CORE_STATE.lastUpdate` |
| **Returns** | `void` |
| **DOM reads** | none |
| **DOM writes** | confScore, confLabel, confFill, cbRSI, cbMACD, cbST, cbLS, cbFR, cbOI |

**Algorithm**:
```
Components: rsiScore, signalRatioScore, stScore, lsScore, frScore, oiScore
Each scored 50-80 range based on thresholds
dirFactor = bullDirs / totalDirs
baseScore = dirFactor × 100
signalBoost = +20 (≥4 signals) | +10 (≥2) | 0
finalScore = round(clamp(0, 100, bull>bear ? base+boost : bull<bear ? base-boost : base))
```

### `computePredatorState()`
**File**: `public/js/core/events.js`  
**Category**: EVENTS — AI veto state machine

| | |
|---|---|
| **Params** | none |
| **Reads** | `BM.volRegime`, `BM.lossStreak`, `BM.riskState`, `BM.mtf.score`, `BM.confluenceScore`, `_SAFETY.dataStalled` |

| **Returns** | `void` |
| **DOM writes** | pred-sleep, pred-hunt, pred-kill (pill styles) |

**State machine**:
```
dataStall           → SLEEP / DATA_STALL
volRegime=EXTREME   → SLEEP / VOL_EXTREME

riskState=RISK_OFF  → HUNT / RISK_OFF
riskState=CHOP      → HUNT / CHOP
volRegime=HIGH      → HUNT / VOL_HIGH
mtf.score<35        → HUNT / MTF_MISALIGN
confluenceScore<40  → HUNT / SCORE_LOW
(else)              → KILL / OK
```

**File**: `public/js/trading/autotrade.js`

| | |
|---|---|
| **Params** | none |

| **Returns** | `void` |

**Execution sequence**:
```
1. computePredatorState()
2. Guard: AT.running → return
3. AT.running = true
4. try:
   a. isDataOkForAutoTrade() → guard
   b. Multi-symbol scan or single-symbol
   c. checkATConditions() → guard (allOk)
   d. AT_BLOCK log if !allOk → return
   e. Guard: AT.enabled → return
   f. Guard: direction clear → return
   g. AT_SIGNAL log
   h. computeFusionDecision() → guard (NO_TRADE → set BlockReason, return)
   i. placeAutoTrade(side, cond)
5. finally: AT.running = false
```

**File**: `public/js/trading/autotrade.js`

| | |
|---|---|
| **Params** | `side` ('LONG'\|'SHORT'), `cond` (from checkATConditions), `_sym?` (override), `_price?` (override) |

| **Returns** | `void` |
| **DOM reads** | none (uses buildExecSnapshot) |

**Veto gates (in order)**:
1. Kill switch active
2. Live exec already in flight
3. Protect mode active
4. DSL mode not set → auto-set 'atr'
5. PREDATOR state ≠ KILL
6. WR Filter (hour winrate below min)
7. buildExecSnapshot returns null
8. Price invalid
9. Dedup (same symbol already open)
10. Margin check (demoBalance < size)

**Sizing chain**:
```
base size (from snapshot)
× fusionMult (LARGE=1.75, MEDIUM=1.35, SMALL=1.0)
× convictionDangerMult (BM.convictionMult)
× Level5 sizing (BM.positionSizing.finalMult if BM.adapt.enabled)
× Adaptive sizing (BM.adaptive.sizeMult if BM.adaptive.enabled)
clamped [0.5×base, 1.6×base]
```

**Position object created** (demo):
```javascript
{
  id: Date.now(), side, sym, entry, size: adaptFinalSize, lev,
  tp, sl, liqPrice, pnl: 0, qty, margin, tpPnl, slPnl,
  autoTrade: true, openTs: Date.now(),

  sizeBase, sizeFinal, sizeMult, adaptSizeMult,
```

## 3. EXECUTION FLOW — COMPLETE AT CHAIN

```
[30s interval]
    │
    ▼
runAutoTradeCheck()
    ├─ computePredatorState()           → PREDATOR.state updated
    ├─ Guard: AT.running                → skip if locked
    ├─ AT.running = true
    │
    ├─ isDataOkForAutoTrade()           → false? return
    ├─ checkATConditions()              → {allOk, isBull, isBear, score, ...}
    │   ├─ if !allOk → AT_BLOCK log    → return
    │   ├─ if !AT.enabled              → return
    │   └─ if no clear direction       → return
    │
    ├─ computeFusionDecision()          → {dir, decision, confidence, ...}
    │   └─ if NO_TRADE                 → BlockReason.set(), return
    │
    └─ placeAutoTrade(side, cond)
        ├─ Kill switch check
        ├─ Live exec dedup
        ├─ Protect mode check
        ├─ DSL mode guard
        ├─ PREDATOR veto (state≠KILL)
        ├─ WR Filter (hour winrate)
        ├─ buildExecSnapshot()         → null? return
        ├─ Price validity
        ├─ Dedup (same symbol open)
        ├─ Sizing chain (5 multipliers, clamped)
        ├─ Margin check
        │
        ├─ [DEMO] → push position, deduct balance, DSL attach
        └─ [LIVE] → API call to exchange
            │
            ▼
        scheduleAutoClose(pos)          → 3s monitoring interval
            ├─ TP hit → close
            ├─ DSL hit → close
            ├─ SL hit → close
            ├─ LIQ hit → close
            └─ TTP (watch only)

[3s interval]
    │
    ▼
runDSLBrain()
    ├─ Guard: DSL.enabled, price valid, no stall
    ├─ forEach open position:
    │   ├─ Phase 1: Activation check
    │   ├─ Phase 2: Yellow line + Pivot Right tracking
    │   ├─ Phase 2.5: PL Exit check
    │   ├─ Phase 3: Impulse Validation
    │   └─ Phase 4: Adaptive state (calm→tense→aggressive)
    └─ renderDSLWidget()
```
